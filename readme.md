# Документация: Автоматический обмен сообщениями между чатами

## Назначение программы

Программа автоматизирует обмен сообщениями между двумя веб-чатами:
- **JuicyChat.AI** (вкладка 1)
- **DeepSeek** (вкладка 2)

Скрипт выполняет циклический обмен сообщениями: копирует последнее сообщение из одного чата и отправляет его в другой.

## Требования

### 1. Программное обеспечение
- **Python 3.7+**
- **Google Chrome**
- **Selenium**

### 2. Установка зависимостей
```bash
pip install selenium
```

## Подготовка к запуску

### Шаг 1: Запуск Chrome в режиме отладки
Перед запуском скрипта необходимо запустить Chrome с параметрами отладки:

**Windows:**
```bash
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\selenium_chrome_profile"
```

**macOS/Linux:**
```bash
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222 --user-data-dir="/tmp/chrome_profile"
```

### Шаг 2: Открытие вкладок
В запущенном браузере откройте две вкладки:
1. **Вкладка 1:** (ваш конкретный чат c ботом JuicyChat)
2. **Вкладка 2:** `https://chat.deepseek.com`

### Шаг 3: Авторизация
**Вручную выполните вход** в оба чата. Программа работает с уже авторизованными сессиями.

## ⚙️ Настройки программы

Основные настройки находятся в начале скрипта:

```python
# Основные настройки
WAIT_TIME = 10          # Время ожидания между действиями (секунды)
MAX_CYCLES = 30         # Количество циклов обмена
```

## Запуск программы 

### 1. Запуск

1. Скачайте скрипт из папки src.
2. Запустите его с помощью команды:

```bash
python main.py
```

### 3. Остановка
- **Ctrl+C** в терминале для остановки
- Программа завершит текущий цикл и остановится
